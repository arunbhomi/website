Trail Mapping & Database Development (NepTrails)

Completed detailed trail mapping using GPS data collection, added waypoints, and developed a comprehensive database for trail management and tourism enhancement in Nepal.

This involved integrating collected data into GIS platforms for visualization and analysis, contributing to sustainable trail infrastructure development.

Interactive Earthquake Impact WebGIS

Developed a real-time earthquake mapping platform with filtering by date and magnitude. The platform shows affected features upon selecting earthquake locations using live API data.

This project combines disaster monitoring with spatial analysis, helping stakeholders understand and respond to earthquake impacts effectively.

Authors: Arun Kumar Bhomi and Rabina Poudyal

House Numbering and Searchable Building Web Map

Created an interactive web map featuring digitized buildings with searchable attributes such as owner name, ward, and house number.

The platform supports boundary visualization and detailed building information display, improving urban data accessibility.

Authors: Arun Kumar Bhomi and Rabina Poudyal

Kathmandu Metropolitan City (KMC) Web Portal

Developed a comprehensive web GIS portal including layers for buildings, schools, roads, and cadastral parcels.

Parcel information includes land use zones, GLD roads, and more, with interactive querying capabilities enhancing urban planning and management.

Author: Arun Kumar Bhomi

Landfill Site Selection GIS Analysis

Performed multi-criteria landfill site selection using spatial datasets such as roads, rivers, settlements, LULC, and elevation.

Processed vector and raster data including Euclidean distance calculations and reclassification for suitability mapping.

UAV and LiDAR Remote Sensing Analysis

Utilized UAV to generate orthomosaics, DEMs, topographic maps, and 3D models of terrain and buildings.

Conducted LiDAR surveys for precise 3D area modeling, enabling detailed spatial visualization for planning and research.

Bridge and Route Survey

Participated in field surveys for bridges and routes, including alignment fixing using GNSS and traditional survey instruments.

Supported infrastructure planning by integrating field data into GIS for spatial analysis and route optimization.

Road Survey Data Collection & GIS Integration

Led a team of enumerators collecting detailed road survey data in Kathmandu Metropolitan City.

Processed data to support municipal planning, with visualization and analysis tools developed to manage the road network effectively.

Automated Feature Digitization using AI/ML (UAV Imagery)

Researched and worked towards automating feature extraction from high-resolution UAV imagery using machine learning algorithms.

This approach aims to improve efficiency and accuracy in urban mapping and cadastral data generation.

GIS Application Development with Leaflet

Developed multiple interactive web GIS applications with Leaflet.js, including parcel information systems, building search functions, and earthquake monitoring dashboards.

Integrated spatial data dynamically for user-friendly mapping solutions without the need for API keys or complex setups.

Data Integration and Visualization Projects

Combined various data formats such as GLD AutoCAD maps, cadastral shapefiles, and zoning maps into unified GIS platforms.

Enabled detailed parcel-level information retrieval with clickable maps supporting urban governance and resource management.

Disaster Response Mapping

Contributed to mapping efforts supporting disaster response and resilience, with spatial data aiding rapid decision-making in emergencies.

Google Earth Engine (GEE) Projects

Executed remote sensing analysis using GEE for environmental monitoring, land use/land cover change detection, and quick geospatial assessments.

Developed scripts and workflows that enable efficient processing of large satellite datasets.
