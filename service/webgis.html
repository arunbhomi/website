WebGIS Projects

Interactive Web Mapping applications developed using Leaflet.js and geospatial data integration for public use, urban analysis, and decision-making.

Technologies Used:

  • Leaflet.js
  • GeoJSON
  • Shapefile to GeoJSON Conversion
  • Parcel & Building Data
  • Interactive Feature Queries
  • Routing & Navigation Tools
  • OpenStreetMap Base Layers
  • Web Hosting Platforms
  • HTML, CSS, JavaScript

1. Earthquake Areas and Affected Features

This interactive WebGIS platform visualizes real-time earthquake data using APIs. Users can select earthquakes based on date and magnitude. Clicking on a specific earthquake displays the location and a list of affected features such as nearby buildings, infrastructure, and administrative boundaries.

Developed by: Arun Kumar Bhomi and Rabina Poudyal

2. House Numbering Map

This WebGIS map displays digitized buildings along with boundaries and essential municipal features. It supports interactivity where clicking on buildings reveals information such as owner name, ward number, and house number. The system also allows users to search for buildings based on these attributes, improving urban management and service delivery.

Developed by: Arun Kumar Bhomi and Rabina Poudyal

3. KMC Web Portal

A comprehensive municipal WebGIS platform for Kathmandu Metropolitan City that integrates multiple spatial layers including buildings, roads, schools, and cadastral parcels. On clicking a land parcel, users can access detailed information such as land use zone, GLD road classification, and more. The platform aims to support urban planning and governance through better spatial data access.

Developed by: Arun Kumar Bhomi
