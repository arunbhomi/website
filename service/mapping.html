GIS Mapping Projects

GIS-based spatial applications developed to support urban planning, emergency response, service delivery, and community mapping using interactive maps and open geospatial data.

Technologies Used:

  • ArcMap 10.8
  • QGIS
  • ArcGIS Pro
  • GeoJSON & Shapefiles
  • OpenStreetMap (OSM)
  • Custom Map Layout Design
  • HTML, CSS, JavaScript

1. Earthquake Response Mapping

This WebGIS project displays earthquake locations and identifies nearby affected features. It allows users to filter quakes by date and magnitude and lists surrounding infrastructures and administrative units, supporting rapid disaster response planning.

Developed by: Arun Kumar Bhomi and Rabina Poudyal

2. Kathmandu Metropolitan City Parcel Map

Integrates building, parcel, zoning, and road classification layers for KMC. Users can click a parcel to retrieve land use details and municipal classifications. The system is built to support planners and citizens with parcel-level spatial insight.

Developed by: Arun Kumar Bhomi

3. House Numbering WebGIS

This system links buildings to house numbers and owner details using digitized shapefiles. The searchable interface allows querying by house number, ward, or owner, improving governance and municipal record management.

Developed by: Arun Kumar Bhomi and Rabina Poudyal

4. Women-Focused POI Mapping

A participatory mapping project focusing on healthcare and emergency-related points of interest (POIs) for women. Data was collected using field mapping and OSM tools, aimed at improving access to safe, relevant facilities.

Initiated and Led by: Arun Kumar Bhomi and Rabina Poudyal

5. Road Survey and Routing Application

Using a GeoJSON road network, this app displays optimized routes between two locations on a Leaflet map without requiring an external API key. The application supports route visualization based on municipal road datasets.

Developed by: Arun Kumar Bhomi and Rabina Poudyal

6. Different Map Layouts Based on Needs

Created GIS layouts tailored for specific use cases such as zoning maps, hazard maps, cadastral overlays, and POI access maps. Each layout is optimized for its audience—planners, emergency teams, or citizens—using thoughtful symbology and legends.

Designed by: Arun Kumar Bhomi

7. Other Web Maps

This section includes additional GIS tools built for specific local needs, such as waste complaint visualization, pipeline restoration updates, and map-based feedback platforms for residents.

Projects by: Arun Kumar Bhomi
